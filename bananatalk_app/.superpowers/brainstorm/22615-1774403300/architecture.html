Key Points

Existing (Enhanced)
  • CallManager — add mute/video emit to server
  • WebRTCService — add multi-peer support
  • ChatSocketService — add voiceroom:* events
  • callProvider — add connection state
New Components
  • VoiceRoomManager — room lifecycle
  • voiceRoomProvider — room state
  • CallHistoryService — fetch/store history
  • Call-related localizations (18 langs)
